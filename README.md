# Folium Map Visualization Tool

## 🗺️ 项目简介
本项目是一个基于 Python **Folium** 库的地图可视化工具。它能够读取文本文件中的地理位置数据（POI，如美食、购物、酒店等），并将其标记在交互式地图上，生成 HTML 文件以便在浏览器中查看。

## 📂 文件说明
*   **`temp.py`**: 主程序脚本，负责读取数据并生成地图。
*   **`point.txt`**: 基础坐标点数据。
*   **`美食.txt`, `购物.txt`, `酒店.txt`**: 分类的 POI 数据文件，包含经纬度和名称。
*   **`resultpreclass-2020-04-18(1).txt`**: 预处理或结果数据。

## 🛠️ 技术栈
*   **Python 3.x**
*   **Folium**: 用于生成 Leaflet.js 地图。
*   **Pandas** (可能用于数据处理).

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install folium pandas
```

### 2. 准备数据
确保你的目录下有相应的 `.txt` 数据文件，格式应符合脚本读取要求（通常为 CSV 或特定分隔符格式）。

### 3. 运行脚本
```bash
python temp.py
```

### 4. 查看结果
运行成功后，会生成一个 `.html` 文件（例如 `map.html` 或类似名称），直接用浏览器打开即可查看带有标记的交互式地图。

## 📊 示例效果
脚本会根据输入的不同类别数据（如美食、酒店），在地图上用不同颜色的标记点（Marker）进行展示。
